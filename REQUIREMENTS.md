## 1. システムの目的（Why）

### 1.1 ビジネス目的
**「作業報告書・材料持出表の作成時間を90%削減し、現場の生産性を向上させる」**

#### 現状の課題（As-Is）
- ❌ **Excelの手作業による報告書作成**：1件あたり30-60分
- ❌ **計算ミスのリスク**：時間計算、金額計算を手動で行うため人的ミス発生
- ❌ **過去データの再利用が困難**：材料名を毎回手入力
- ❌ **スマホから入力できない**：Excelは現場での入力に不向き
- ❌ **事務作業の負担増**：現場社員（大竹、豊島、鈴木）が事務作業に時間を取られる

#### 目指す姿（To-Be）
- ✅ **報告書作成時間：3-5分/件**（90%削減）
- ✅ **計算ミスゼロ**：自動計算により人的ミス排除
- ✅ **履歴からワンクリック入力**：過去の材料名を再利用
- ✅ **現場でスマホ入力可能**：作業終了後すぐに報告完了
- ✅ **事務職採用の効果最大化**：事務作業を事務職に委譲し、現場は技術に集中

### 1.2 採用戦略との連動
FITの「2軸採用戦略」における**事務職採用の成功要因**として位置づけられる：

- **事務職のミッション**: 現場の事務作業負担を軽減すること
- **このシステムの役割**: 事務職が効率的に報告書を作成できる環境を提供
- **期待効果**: 
  - 事務職の生産性向上 → 早期戦力化
  - 現場社員の負担減 → 技術業務に集中
  - 豊島社長の管理工数削減 → 経営に集中

- **豊島社長**: 多忙、シンプルで直感的なUIを好む

## 3. システム要件（What）

### 3.1 機能要件

#### 【優先度：最高】Phase 1（2026/01/19完成）

##### 3.1.1 作業報告書機能
**目的**: 作業時間と工賃を自動計算し、報告書を3分で作成可能にする

| 機能ID | 機能名 | 詳細 |
|:---|:---|:---|
| F-001 | 基本情報入力 | 船名、科目、型名、完成月日を入力 |
| F-002 | 作業者選択 | 大竹、豊島、鈴木、内田、新人から選択（複数選択可） |
| F-003 | 時間入力 | 開始時刻・終了時刻を入力（時間:分形式） |
| F-004 | 時間区分選択 | 移動/時間内/時間外/休日をチェックボックスで選択 |
| F-005 | 時間追加 | 同一作業者の時間帯を複数追加可能 |
| F-006 | **自動計算（工賃）** | **平日時間内: 7,000円/h<br>平日時間外: 7,000円/h<br>休日: 8,400円/h<br>移動費: 7,000円/h × 0.8倍** |
| F-007 | 作業者別集計 | 作業者ごとの時間・金額を自動集計 |
| F-008 | 総合計表示 | 全作業者の合計金額を表示 |

**ビジネスルール**:
```
工賃計算ロジック：
- 移動時間: 時間数 × 7,000円 × 0.8
- 時間内: 時間数 × 7,000円
- 時間外: 時間数 × 7,000円
- 休日: 時間数 × 8,400円

作業者ごとの合計 = 移動 + 時間内 + 時間外 + 休日
総合計 = 全作業者の合計の総和
```

##### 3.1.2 材料持出表機能
**目的**: 材料の仕入・売値を管理し、粗利を可視化する

| 機能ID | 機能名 | 詳細 |
|:---|:---|:---|
| F-101 | 基本情報入力 | 船名、科目、型名、完成月日を入力 |
| F-102 | 材料行追加/削除 | 材料を動的に追加・削除 |
| F-103 | **履歴学習型商品名入力** | **過去入力した商品名を自動保存し、次回から選択可能（LocalStorage活用）** |
| F-104 | 材料詳細入力 | 日付、商品名、型式、在庫有無、仕入先、数量、単価、送料 |
| F-105 | 仕入先選択 | モノタロウ、アマゾン、ハートストック、JRC、その他から選択 |
| F-106 | **自動計算（金額）** | **仕入合計 = 数量 × 仕入単価<br>売値合計 = 数量 × 売値単価** |
| F-107 | **粗利計算** | **粗利 = 売値合計 - 仕入合計 - 送料合計** |
| F-108 | 合計金額表示 | 仕入合計、売値合計、送料合計、粗利を表示 |

**ビジネスルール**:
```
材料計算ロジック：
- 仕入合計 = 数量 × 仕入単価
- 売値合計 = 数量 × 売値単価
- 粗利 = 売値合計 - 仕入合計 - 送料合計

履歴学習：
- 商品名入力完了時、LocalStorageに自動保存
- 最大50件まで保存
- 保存履歴は入力時にComboboxで選択可能
```

## 6. 3月の契約交渉への活用戦略

### 6.1 コスト削減効果の試算

```
【現状】
- 報告書作成: 30分/件 × 時給換算3,000円 = 1,500円/件
- 月間作業件数: 20件と仮定
- 月間コスト: 1,500円 × 20件 = 30,000円

【システム導入後】
- 報告書作成: 5分/件 × 時給換算3,000円 = 250円/件
- 月間コスト: 250円 × 20件 = 5,000円

【削減効果】
- 月間削減額: 25,000円
- 年間削減額: 300,000円
- ROI: 運用費60,000円/年 → 投資回収期間: 2.4ヶ月
```

**豊島社長への提案メッセージ（3月）**:

> 「1月に無償提供した業務効率化アプリにより、報告書作成時間が90%削減され、年間30万円のコスト削減を実現しました。これは事務職採用の成功要因となり、現場社員が技術業務に集中できる環境が整いました。
> 
> 今後、採用メディア戦略（SNS運用・求人広告）と合わせて、月額○万円の契約とさせていただけますと、さらなる業務効率化と採用成功を支援させていただきます。」
